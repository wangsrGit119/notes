## 简易安装步骤

注意事项：下面paddle对应的 protobuf和onnx的版本
```
protobuf==3.20.0
onnx==1.11.0
paddlepaddle==2.4.2
```

1.安装PaddlePaddle
2.安装验证

```
import paddle    
print(paddle.__version__)
paddle.utils.run_check()

```
3.安装 PaddleClas 及其 Python 依赖库

```
pip install paddleclas

```
