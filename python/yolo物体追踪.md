

## get start

git clone https://github.com/mikel-brostrom/yolo_tracking

cd yolo_tracking

如果您想运行 YOLOv8、YOLO-NAS 或 YOLOX 示例：

git clone https://github.com/mikel-brostrom/yolo_tracking.git
pip install -v -e .
但如果您只想导入跟踪模块，您可以简单地：

pip install boxmot
